# 🚀 User Management System

A full-stack CRUD application built using **Node.js, Express, MongoDB, and EJS** with a modern UI using **Tailwind CSS**.

---

## 📌 Features

- ✅ Create User
- 📖 Read / View Users
- ✏️ Update User
- ❌ Delete User
- 🎨 Clean and responsive UI (Tailwind CSS)
- ⚡ Server-side validation
- 🛡 Error handling

---

## 🛠 Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Frontend:** EJS, Tailwind CSS
